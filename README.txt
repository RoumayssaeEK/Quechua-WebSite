# QUECHUA - Site Web de Karaoké

**Développé par : Roumayssae EL KENZ**



Structure des Fichiers:

 config/
- **`database.php`** : Configuration de la base de données (connexion MySQL)

 includes/
- footer.php : Pied de page commun à toutes les pages
- header.php : En-tête avec meta tags, CSS et scripts
- navigation.php : Menu de navigation principal

 lang/
- es.php : Fichier de traduction espagnole
- fr.php : Fichier de traduction française

media/
- audios/: Stockage des fichiers audio des chansons
- images : Images du site (logos, illustrations, etc.)
- karaoke/ : Fichiers spécifiques au karaoké (vidéos, pistes, etc.)

 Pages Principales
- admin.php : Interface d'administration du site
- chanson.php : Page détaillée d'une chanson
- contact.php : Formulaire de contact
- functions.php : Fonctions utilitaires et helpers
- index.php: Page d'accueil du site
- interprete.php: Page détaillée d'un interprète/artiste
- like_chanson.php : Système de likes pour les chansons
- liste_chansons.php : Liste/catalogue des chansons disponibles
- liste_interpretes.php: Liste des interprètes/artistes
- login.php : Page de connexion utilisateur
-logout.php : Script de déconnexion et destruction de session
-traitement_contact.php : Traitement des données du formulaire de contact
